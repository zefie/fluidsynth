# FluidSynth

For information on FluidSynth, please visit [https://github.com/fluidsynth/](https://github.com/FluidSynth/fluidsynth).

This repo contains zefie's modifications, which include the following two patches:
1) Allow compilation with emscripten: [69c33ac](https://github.com/zefie/fluidsynth/commit/69c33ac87ff276b69772d2cecb634da02145131a)
2) Make FluidSynth more tolerant to DLS bank errors: [1bd6b4a](https://github.com/zefie/fluidsynth/commit/1bd6b4aa78891af65cf5f26bf0f54aa1127fd78d)
